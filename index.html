Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans: ok, so 1st innerHTML is used to get all the Html structure from Dom. Like we have target a button then we can see the button html structure by this innerHTML property. And then 2nd thing is innerText and textContent this is almost same things but the main difference is, with innerText we get only the main text from an element. But with the textContent we get other child text and main text if is it there any child text.

Question-4: What is event bubbling in the DOM?

Ans: event bubbling is like a loop in Dom. When we run a event in dom then it start actually from the root element and go to until it find the child element i mean targeted element. After find the child then it start again come back to the root. So this things is bubbling in Dom. But i like to consider this bubbling as loop for my personal understanding.
